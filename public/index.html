pixhub-rs

pixiv 与 douyin 资源后端 API
运行中
⚡ Rust · Axum · SQLite
Pixiv 图片4 端点
GET
/pixiv/artworks/random
随机获取一张 pixiv 图片信息
GET
/pixiv/artworks/info?list={文件名}
根据文件名查询 pixiv 图片信息
GET
/pixiv/artworks/view
直接查看随机 pixiv 图片
GET
/pixiv/artworks/file/{文件名}
获取 pixiv 图片文件
Plus 视频5 端点
GET
/plus/artworks/random
随机获取一个视频信息
GET
/plus/artworks/info?list={作者}
根据作者名或作者 ID 查询视频信息
GET
/plus/artworks/info?list=all
获取所有作者列表
GET
/plus/artworks/view
直接查看随机视频
GET
/plus/artworks/file/{文件名}
获取视频文件
